### brains
系统通知配置






























用户属性不直接与第三方系统的属性进行映射
集成第三方系统后，由第三方系统进行映射
如果集成了多个三方系统，每次登录都覆盖第三方系统的数据
系统属性，全局一份，有多分属性
用户、属性、属性值，单独存储
